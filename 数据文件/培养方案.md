## 字段说明

| 字段 | 含义 |
| --- | --- |
| 体系 | 培养方案中的课程体系类别，如公共基础、学科基础、专业知识、工作技能等 |
| 选课组 | 课程所在选课组；截图中多数为空，故保留为空 |
| 课程编号 | 官方课程编号 |
| 课程名称 | 课程名称 |
| 课程性质 | 课程所属性质，如公共基础必修课、专业知识选修课等 |
| 课程性 | 必修 / 选修 / 外语课等 |
| 学分 | 课程学分 |
| 讲课学时 | 理论讲授学时 |
| 实践学时 | 实验、实践、训练等学时 |
| 总学时 | 课程总学时 |
| 开设学期 | 推荐开设学期 |
