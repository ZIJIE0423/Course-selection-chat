# 官方选课信息结构化数据（评价参考适配版）

> 数据用途：用于智能选课系统的结构化课程数据库导入与检索测试。
> 适配说明：本文件沿用“官方选课信息结构化数据”的字段格式；课程名称与授课教师参考上传的《选课评价参考》中出现的老师/课程组合；选课号、课程编号、校区、上课院系等字段为项目测试所需的结构化填补数据，并非真实教务系统导出结果。

- 适配记录数：130 条
- 字段：学年学期、选课号、校区、课程编号、课程名称、课程层面、授课教师、上课院系、来源截图
- 数据分层建议：本文件可作为 `official_structured_db` 的测试数据；学生评价原文仍建议单独作为 `student_review_rag` 非官方经验库。

## 字段说明

| 字段 | 含义 |
|---|---|
| 学年学期 | 课程所属学年学期，如 2025-2026-3 |
| 选课号 | 学生选课时使用的教学班/选课编号 |
| 校区 | 课程开设校区 |
| 课程编号 | 课程代码/课程编号，按课程类别模拟生成 |
| 课程名称 | 课程名称，优先采用评价参考中出现的课程名称并规范化 |
| 课程层面 | 公共基础课、专业类课程、通识教育课程等 |
| 授课教师 | 评价参考中出现的授课教师 |
| 上课院系 | 课程归属或面向院系，按课程类别生成 |
| 来源截图 | 沿用原官方选课信息字段名；此处记录适配来源编号，便于回溯到评价参考条目 |
